# Week 3: Tooling, Culture and Financial Literacy

## Monday: Web3 Wallets, dApps and Security for Creatives

Objectives:
- Gain confidence in safely interacting with Web3 tools
- Learn types of wallets: Metamask, Trust, Ledger
- Learn how to connect to dApps
- Understand scams, phishing, rugpull awareness
- Learn backup and recovery best practices

## Deliverable: Testnet Transaction

Connect to Sepolia testnet and simulate a transaction.

Steps:
1. Set up a MetaMask wallet or similar
2. Connect to Sepolia testnet
3. Get testnet ETH from a faucet
4. Make a test transaction
5. Document the process:
   - Screenshots of wallet setup
   - Transaction hash
   - What you learned about gas fees
   - Security practices you followed
6. Add your documentation to your "My Web3 Journey" repo

## Wednesday: Tokenomics for Creatives plus Web3 Culture

Objectives:
- Understand what tokenomics is
- Learn utility vs governance vs security tokens
- Understand pump and dump cycles and why storytelling drives value
- Learn meme culture, CT, narratives, degen behavior
- Understand why memes onboard more than whitepapers

## Deliverable: Token Concept Pitch

Create a sample token concept in groups and pitch with a meme.

Your token concept includes:
- Token name
- Purpose
- Utility or governance function
- Target audience
- Meme or visual representation

Document your group work:
- Group members
- Token concept details
- Meme or visual
- Pitch summary
- Add to your "My Web3 Journey" repo

## Friday: Web3 Marketplaces and Creative Revenue Streams

Objectives:
- Learn how blockchain powers monetization
- Understand NFT marketplaces: OpenSea, Foundation
- Learn about royalties and secondary markets
- Understand on-chain vs off-chain storage: IPFS, Arweave
- Learn earning through DAOs, grants, bounty boards
- Understand Web3 jargon simplified: gm, wagmi, anon, fomo

## Deliverable: Web3 Earning Paths

List 3 Web3 earning paths for your creative type.

For each path, document:
- What it is
- How it works
- Requirements or skills needed
- Examples of people doing it
- Potential earnings
- Steps to get started

Add your analysis to your "My Web3 Journey" repo.

## How to Submit Your Assignment

Follow these steps:

1. Update your fork of this repository
2. Create a new branch for your submission:
   git checkout -b week-3-assignment

3. Add your work inside the submissions folder:
   submissions/
       └── week-3/
           └── your-name/
               ├── testnet-transaction.md
               ├── token-concept-pitch.md
               └── earning-paths.md

4. Include links to your "My Web3 Journey" repo where relevant

5. Commit and push your changes:
   git add .
   git commit -m "chore: submit week 3 assignment"
   git push origin week-3-assignment

6. Create a Pull Request to the main repository. Your PR includes:
   - A brief summary of what you learned
   - Links to your deliverables in your personal repo

7. Post at least one reflection on social media about your Week 3 learnings

## Evaluation Criteria

Submissions are evaluated based on:
- Completeness of all three deliverables
- Practical application of Web3 tools
- Clarity of documentation
- Consistency in following submission structure
- Creativity in token concept and meme
